# Compiler IR Visualizer

A frontend compiler lab that parses expressions, visualizes AST structure, and emits optimized three-address IR.

## Highlights
- In-browser expression tokenization and parsing.
- AST visualization on Canvas.
- Three-address code generation.
- Constant-folding optimization pass.

## Stack
- HTML5 Canvas
- Vanilla JavaScript
- Responsive CSS interface

## Run
Open `index.html` in your browser.
